
# Survive the Internet (CSC209 A3)

A multiplayer party game inspired by **Survive the Internet**.

This page is meant to explain what the game is, how it is played, and what someone should expect when joining a match.

## What the game is

Survive the Internet is a social game built around funny writing, unexpected twists, and group reactions. Each player is given chances to answer prompts, respond to situations, and react to the choices made by the other players.

The fun of the game comes from seeing ordinary answers turned into something more dramatic, ridiculous, or embarrassing. A response that looks harmless at first can become much funnier once it is shown in a different way later in the round.

This version is designed as a small group game for **3 to 5 players**.

## Goal of the game

The goal is to be the funniest player in the group.

Players try to write answers that will lead to the best reactions once they are revealed. Sometimes that means being clever, sometimes it means being chaotic, and sometimes it means setting up another player for a ridiculous moment.

By the end of the game, the player who creates the strongest overall impression through their answers and choices is the winner.

## How a game usually works

A full game is played in rounds.

During a round, players will usually:

1. Read a prompt or short situation.
2. Write a response.
3. Wait while the other players finish.
4. See responses appear in a new or altered context.
5. React to the outcome and move on to the next round.


## Basic rules

- The game is meant for **3 to 5 players**.
- Every player joins the same match and plays through the same rounds.
- Players should use a unique username.
- Everyone should wait for the current round to finish before expecting the next step.
- If too many players leave, the game may end early.

## Winning

The winner is the player who performs best across the full set of rounds.

That usually means consistently giving answers that create strong reactions and memorable moments.
